Arquitetura dos Computadores

O que é a arquitetura de computadores? Para que o conceito fique mais claro, é bom entender que arquitetura se refere a um projeto de uma construção. Ela vai adotar uma série de ações para organizar e criar espaços que possam dar condições a diferentes atividades humanas.
